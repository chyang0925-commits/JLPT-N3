# JLPT N3 학습앱

다락원 『JLPT 한권으로 끝내기 N3』 문자·어휘 파트(한자읽기)를 기반으로 제작한 개인 학습용 웹앱입니다. N4 학습앱과 동일한 구조로 만들어졌으며, 설치 없이 브라우저에서 바로 실행되고 휴대폰·태블릿·PC 어디서나 사용할 수 있습니다.

## 구성

| 과 | 영역 | 내용 |
|---|---|---|
| 1과 | 한자읽기 | 기출어휘 2016~2024(89개) · 확인문제 1~7(총 70문항) |

현재는 1과만 수록되어 있으며, 교재 사진이 추가로 촬영되는 대로 2과 이상이 계속 추가될 예정입니다. (내부적으로 `V2`/`P2` 데이터와 `LESSONS` 배열에 한 줄만 추가하면 되도록 구조화되어 있습니다.)

주요 기능: 어휘 플래시카드(암기 힌트 포함), 퀴즈 3종, 실전 확인문제 풀이(한국어 해석 제공), 오답 자동 저장 및 복습.

### 퀴즈 3종 안내 (N4와 다른 점)

N4 학습앱은 퀴즈③이 "히라가나→한자"였지만, N3 학습앱은 HIRO의 요청에 따라 방향을 바꿔 다음 3종으로 구성했습니다.

1. **퀴즈①** 히라가나 → 한글 (히라가나를 보고 뜻 고르기)
2. **퀴즈②** 한자 → 한글 (한자를 보고 뜻 고르기)
3. **퀴즈③** 한자 → 히라가나 (한자를 보고 올바른 읽기 고르기) — N4와 반대 방향

## 실행 방법

`index.html` 파일 하나로 동작합니다. 별도의 빌드나 설치가 필요 없습니다.

- **로컬 실행**: `index.html`을 더블클릭하여 브라우저로 열기 (인터넷 연결 필요 — React를 CDN에서 불러옵니다)
- **GitHub Pages 호스팅**: N4 학습앱 README의 안내와 동일한 방식으로 진행하면 됩니다 (Private 저장소 권장, `index.html`만 업로드해도 전체 데이터·로직이 내장되어 바로 동작).

## 파일 안내

| 파일 | 용도 |

## 데이터 출처 및 검증 이력

- 1과 기출어휘(89개)·확인문제 7세트(70문항)는 다락원 N3 교재 실물 사진(p.15~25)을 직접 읽어 전수 전사했습니다.
- 각 확인문제 페이지 하단에 인쇄된 정답 표(예: `답 1② 2③ …`)와 대조했으나, 사진 해상도·회전 상태상 작은 동그라미 숫자(①②③④)를 완벽히 판독하기 어려운 경우가 있어, 그런 항목은 **밑줄 단어의 표준 일본어 읽기(사전적으로 명백한 정답)** 를 기준으로 정답 인덱스를 확정했습니다. 문제 유형 자체가 "밑줄 친 단어의 올바른 히라가나 읽기 고르기"이므로 정답은 항상 언어학적으로 유일하며, 이 방식이 안전합니다.
- 어휘 목록(연도별 기출어휘)과 확인문제에 등장하는 단어 읽기가 서로 높은 비율로 교차 일치함을 확인하여 전사 정확도를 재검증했습니다.
- 한국어 번역(전체 해석)과 암기 힌트(한자 파자)는 이 앱을 만들며 새로 작성한 것으로, 교재 원문에는 없습니다.

## 유의 사항

- 본 앱은 개인 학습 목적으로 제작되었습니다. 교재 내용의 권리는 출판사(다락원)에 있으며, 상업적 이용 및 무단 배포를 금합니다.
- 오답 기록은 브라우저(localStorage)에 저장되므로 기기·브라우저를 바꾸면 초기화됩니다.
- 확인문제 몇 개 항목은 정답 표 숫자 판독이 애매했던 부분이 있어 HIRO가 실물 교재로 한 번 더 확인해 주시면 좋습니다 (자세한 목록은 작업 완료 보고 참고).
